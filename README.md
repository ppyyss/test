# Five-in-a-Row Game (Gomoku)

A simple terminal-based implementation of the classic Gomoku (Five in a Row) game on a 15x15 board.

## How to Play
- **Run**: `python five_in_a_row.py`
- **Turns**: Players alternate placing 'X' and 'O'
- **Win**: First to get five consecutive marks in a row (horizontal, vertical, diagonal)
- **Input**: Enter row and column (1-15, e.g., `8 8`)

## Features
- 15×15 grid
- Automatic win detection across 4 directions
- Simple terminal UI with '.' for empty cells
- No dependencies (pure Python)
